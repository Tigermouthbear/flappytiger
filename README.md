# FlappyTiger
The flappybird clone you definately dont need. This is my first game written in C++ using glfw.

## Installing
Linux:
```
git clone https://github.com/Tigermouthbear/flappytiger.git
cd flappytiger
mkdir build && cd build
cmake ..
make
sudo make install
```
